Challenge 5: Accessability Statement

There are several issues that this website could have with accessability. To start off, the chess board would be very hard to describe in table format, since it is for a visual and not data or even usable for the game. It would be better if there was alt text to describe format of the chess board as an image. Since this is not an image, it would likely just explain each line instead of explaining it is a unusable chess board. Another concern with accessibility would be the colors I chose for my chess pieces, since I picked pink and green this could be bad for red/green colorblind individuals, instead I could chose colors that would be easily distinguishable or pattern disparities between the two teams.
